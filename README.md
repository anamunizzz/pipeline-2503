# 🚀 Todo API com CI

Este projeto é uma API simples de gerenciamento de tarefas (To-Do), desenvolvida em Python com Flask, com testes automatizados e integração contínua utilizando GitHub Actions.

## 📌 Funcionalidades

* Criar tarefas
* Listar tarefas
* Marcar tarefas como concluídas

## 🛠️ Tecnologias utilizadas

* Python
* Flask
* Pytest
* GitHub Actions (CI)

## 🧪 Testes automatizados

O projeto possui testes automatizados utilizando Pytest para garantir o funcionamento das principais funcionalidades da API.

Para rodar os testes:

```bash
python -m pytest
```

## ▶️ Como executar o projeto

1. Instale as dependências:

```bash
py -m pip install -r requirements.txt
```

2. Execute a aplicação:

```bash
python app.py
```

3. Acesse no navegador:

```
http://127.0.0.1:5000/tasks
```

## 🔁 Integração Contínua

O projeto utiliza GitHub Actions para rodar automaticamente os testes a cada alteração no repositório, garantindo maior qualidade e confiabilidade do código.

## 💡 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Desenvolvimento de APIs
* Testes automatizados
* Integração contínua (CI)
